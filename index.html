ขอ CPA Certificate จากกรมสรรพากร
ก่อนเริ่มใช้งานระบบ Host to Host กับกรมสรรพากร ผู้ประกอบการต้องยื่นเรื่องขอรับ CPA Certificate (Client Certificate) เพื่อพิสูจน์ตัวตนในการส่งข้อมูลอิเล็กทรอนิกส์
1
ยื่นหนังสือแจ้งความประสงค์
ติดต่อกองบริหารการเสียภาษีทางอิเล็กทรอนิกส์ กรมสรรพากร ชั้น 21 อาคารกรมสรรพากร
📞 โทร. 02 272 9771 หรือ etax.rd.go.th
2
เตรียมเอกสารที่ต้องใช้
📄 หนังสือแจ้งความประสงค์ขอเชื่อมต่อระบบ Host to Host (บ.อ.01)
📄 สำเนาหนังสือรับรองบริษัท (ไม่เกิน 6 เดือน)
📄 สำเนาบัตรประชาชนผู้มีอำนาจลงนาม
📄 ข้อมูล IP Address และ Endpoint ของระบบ
📄 SSL Certificate ของ Server
3
รอผลการพิจารณา
กรมสรรพากรจะพิจารณาคุณสมบัติและแจ้งผลทางอีเมลที่ลงทะเบียนไว้ภายใน 5–10 วันทำการ
เมื่อผ่านการพิจารณา กรมสรรพากรจะออก CPA (UAT) ให้เพื่อทดสอบการส่งข้อมูล
4
ดาวน์โหลด CPA (UAT) และตั้งค่าระบบ
ดาวน์โหลด CPA Certificate สำหรับทดสอบจากอีเมลที่กรมสรรพากรส่งให้ นำไป Register ในระบบเพื่อเริ่มทดสอบส่งข้อมูล 4 ประเภท ได้แก่ TIV, DBN, CDN, RCT
💡
หลังจากได้รับ CPA (UAT) จากกรมสรรพากรแล้ว กด "Next" เพื่อไปยังขั้นตอน Register CPA เข้าระบบ และเริ่มทดสอบการส่งข้อมูล
✅ ได้รับ CPA (UAT) จากกรมสรรพากรแล้ว กรอกข้อมูลด้านล่างเพื่อ Register เข้าระบบ
